# yi.wuaishare.cn 易学工具站开发需求文档

> 面向 Codex / 开发者使用。本文档整理自本轮对话的完整规划背景，用于快速启动 `yi.wuaishare.cn` 子站建设，首期功能为「五行穿衣指南查询工具」，后续扩展到八字、周易、六爻、梅花易数、紫微斗数、风水、姓名学等东方传统文化工具。

---

## 1. 项目背景

当前主站为：

- 吾爱分享：`www.wuaishare.cn`
- 我爱分享：`v.wuaishare.cn`
- 吾爱分享社区：`sns.wuaishare.cn`

在 `www.wuaishare.cn` 上已测试发布过「五行穿衣指南」类文章，初步确认该方向适合做日更长尾搜索内容和工具化产品。

已测试内容包括：

- 今日五行穿衣指南文章：`https://www.wuaishare.cn/9086.html`
- 明日五行穿衣指南文章：`https://www.wuaishare.cn/9087.html`
- 已创建分类：`五行穿衣`
  - category ID：`1419`
  - slug：`wuxing-chuanyi`
- 已创建部分标签：
  - `1420`：今日五行穿衣
  - `1421`：五行穿衣指南
  - `1422`：每日穿衣颜色
  - `1423`：今日穿衣颜色
  - `1424`：今日大吉色
  - `1425`：明日五行穿衣
  - `1426`：明日穿衣颜色
  - `1427`：明日大吉色

测试后得出的结论：

1. 「五行穿衣」底层逻辑高度模板化，适合自动化生产。
2. 单纯日更文章可以做，但更有长期价值的是做成一个「可查询工具」。
3. 工具页应默认显示当天五行穿衣指南，并支持前一天、后一天、任意日期查询。
4. 日更文章可继续用于搜索流量入口，工具页负责用户留存和复访。
5. 该方向后续可扩展为一个东方传统文化工具站。

## 2. 子站定位

### 2.1 子域名

确定使用：

```text
yi.wuaishare.cn
```

选择 `yi` 的原因：

- 代表易学、周易、易经、易理、易数。
- 比 `xuan` 更垂直，更适合东方传统文化工具。
- 与后续规划的八字、周易、六爻、梅花易数、紫微斗数、风水、姓名学等方向一致。
- 星座、塔罗等西方占卜体系暂不放入该站，未来可单独用其它域名/子站承接。

### 2.2 站点名称候选

两个名称都可用：

#### 方案 A：易知阁

推荐作为主品牌名。

优点：

- 简洁、正统、有品牌感。
- 「易」对应易学，「知」对应查询、认知、理解，「阁」有传统文化氛围。
- 适合长期扩展为东方传统文化工具站。

#### 方案 B：吾爱易学

适合作为副标题或品牌关联名。

优点：

- 与主站「吾爱分享」有品牌关联。
- 用户容易理解站点方向。
- 适合用于 Logo 下方说明或 footer。

### 2.3 推荐最终命名组合

```text
站点名称：易知阁
品牌副标题：吾爱易学工具站
域名：yi.wuaishare.cn
```

## 4. 首期 MVP：五行穿衣指南查询工具

### 4.1 页面目标

做一个类似万年历的五行穿衣查询工具。

用户打开页面后：

1. 默认显示当天五行穿衣指南。
2. 可点击「前一天」「后一天」。
3. 可选择任意日期查询。
4. 显示该日期的公历、农历、星期、干支、日五行、大吉色、次吉色、平平色、慎用色、不宜色、穿搭建议。
5. 根据日期和日五行自动选择对应封面图/色系图。

### 4.2 工具页 URL

主工具页：

```text
https://yi.wuaishare.cn/wuxing-chuanyi/
```

日期查询页建议：

```text
https://yi.wuaishare.cn/wuxing-chuanyi/2026-05-13/
```

如果首期实现难度较高，可先使用 query 参数版本：

```text
https://yi.wuaishare.cn/wuxing-chuanyi/?date=2026-05-13
```

但长期 SEO 更建议做成日期路径。

### 4.3 页面标题规则

#### 工具首页标题

```text
五行穿衣指南查询 - 今日大吉色、每日穿衣颜色参考
```

#### 日期页标题

```text
2026年5月13日五行穿衣指南：今日大吉色与穿衣颜色
```

#### 日更文章标题策略

前一晚发布：

```text
明日五行穿衣指南（2026年5月13日）：大吉色、次吉色与穿搭建议
```

当天早上更新同一篇文章标题为：

```text
今日五行穿衣指南（2026年5月13日）：大吉色、次吉色与穿搭建议
```

不要同一天发两篇高度重复文章。推荐策略是同一篇文章在前一晚以「明日」预热，第二天清晨改成「今日」。

## 5. 五行穿衣计算逻辑

### 5.1 基础流程

输入：公历日期。

输出：

1. 公历日期
2. 农历日期
3. 星期
4. 年干支
5. 月干支
6. 日干支
7. 日五行
8. 五行颜色推荐
9. 穿搭建议

### 5.2 日五行判断

首期可采用「日地支五行」口径。

地支五行对应：

```json
{
  "子": "水",
  "亥": "水",
  "寅": "木",
  "卯": "木",
  "巳": "火",
  "午": "火",
  "申": "金",
  "酉": "金",
  "辰": "土",
  "戌": "土",
  "丑": "土",
  "未": "土"
}
```

示例：

- 2026-05-12：丙戌日，戌属土，所以日五行为土。
- 2026-05-13：丁亥日，亥属水，所以日五行为水。

注意：不同黄历/万年历可能存在口径差异。首期需固定一种口径，并在页面说明「按常见万年历干支口径」。

### 5.3 五行相生

```text
木生火
火生土
土生金
金生水
水生木
```

### 5.4 五行相克

```text
木克土
土克水
水克火
火克金
金克木
```

### 5.5 颜色对应

```json
{
  "木": ["绿色", "青色", "浅绿"],
  "火": ["红色", "粉色", "紫色", "橙红"],
  "土": ["黄色", "咖色", "棕色", "米黄", "驼色"],
  "金": ["白色", "银色", "灰色", "金色", "米白"],
  "水": ["黑色", "蓝色", "深蓝"]
}
```

### 5.6 推荐等级规则

固定采用以下规则：

| 等级 | 规则 |
|---|---|
| 大吉色 | 当日五行所生的五行颜色 |
| 次吉色 | 与当日五行相同的颜色 |
| 平平色 | 颜色五行克当日五行 |
| 慎用色 | 颜色五行生当日五行 |
| 不宜色 | 当日五行克颜色五行 |

示例：水日。

- 水生木 → 大吉色：木系，绿色、青色、浅绿。
- 水同水 → 次吉色：水系，黑色、蓝色、深蓝。
- 土克水 → 平平色：土系，黄色、咖色、棕色、米黄、驼色。
- 金生水 → 慎用色：金系，白色、银色、灰色、金色、米白。
- 水克火 → 不宜色：火系，红色、粉色、紫色、橙红。

### 5.7 五种核心模板

#### 木日

- 大吉色：火系，红色、粉色、紫色、橙红
- 次吉色：木系，绿色、青色、浅绿
- 平平色：金系或土系需按规则实现，不建议手写死，代码按相生相克计算
- 慎用色：水系
- 不宜色：土系或金系需按规则计算

建议代码统一按关系表生成，避免人工写错。

#### 火日

- 大吉色：土系
- 次吉色：火系

#### 土日

- 大吉色：金系
- 次吉色：土系

#### 金日

- 大吉色：水系
- 次吉色：金系

#### 水日

- 大吉色：木系
- 次吉色：水系

```text
五行穿衣推荐卡片图，{季节}，{人群}，{场景}穿搭，画面为服装平铺或无脸半身穿搭，不出现文字，风格干净实用，适合工具网页卡片，4:3 或 1:1 构图。
```

### 8.7 视觉注意事项

本轮测试中出现过两个问题：

1. 生成图容易偏女性服装。
2. 为了规避女性化，又容易偏纯男装。

后续规则：

- Hero 图统一做中性通用，不突出男装或女装。
- 推荐卡片明确分男装、女装、中性，避免模型自行偏向。
- Hero 图避免高跟鞋、手提包、首饰作为画面主体。
- 工具页主视觉更像「颜色指南」而不是「时尚杂志」。
- 可多使用色卡、衣架、衬衫、外套、裤装、鞋履、布料样本。

## 9. 内容生成策略

### 9.1 文章不是无限复制 5 篇

虽然底层五行只有 5 类，但不建议简单复制 5 篇模板。

```text
5 套核心模板 + 多组开头/结尾/场景建议变体
```

1. 今日/明日五行穿衣指南
2. 日期、农历、干支、日五行
3. 今日大吉色
4. 今日次吉色

7. 今日不宜色
8. 实用穿搭方案
9. 五行穿衣颜色对应表
10. 温馨提示

- 10 套开头导语

## 10. 技术实现建议

### 10.1 推荐技术路线

首期可以有两种实现方式：

#### 方案 A：WordPress 子站 + 自定义插件/短代码

适合当前生态。

实现：

- 在 `yi.wuaishare.cn` 搭建独立 WordPress 子站。
- 写一个自定义插件或主题模板。
- 提供 shortcode：`[wuxing_chuanyi]`。
- 页面中嵌入查询工具。
- 后续文章/SEO 页面仍走 WordPress。

#### 方案 B：独立前端应用 + API

适合工具复杂化后。

实现：

- Next.js / Nuxt / Astro 等静态或 SSR 工具站。
- WordPress 只作为内容管理或完全不用 WordPress。
- 更适合做大量日期页和工具交互。

当前建议先用方案 A，速度更快。

### 10.2 核心模块

开发时建议拆成：

```text
/calendar.ts           日期、农历、干支计算
/wuxing.ts             五行相生相克、颜色推荐
/content-template.ts   文案模板
/images.ts             素材映射
/routes.ts             日期 URL 处理
/seo.ts                title、description、canonical
```

### 10.3 日期页 SEO

每个日期页应输出：

- title
- meta description
- canonical
- Open Graph title
- Open Graph image
- JSON-LD Breadcrumb

示例 title：

```text
2026年5月13日五行穿衣指南：今日大吉色与穿衣颜色
```

示例 description：

```text
查询2026年5月13日五行穿衣指南，今日干支、日五行、大吉色、次吉色、不宜色及通勤、休闲、正式场合穿搭建议。
```

### 10.4 sitemap

后续可生成未来 365 天日期页 sitemap。

但需注意：如果日期页全部动态生成，内容相似度较高，需要保证每页有足够差异化内容或先限制索引范围。

建议首期：

- 工具首页可索引。
- 今日/明日/近 30 天日期页可索引。
- 太远未来日期页先不主动提交 sitemap。

## 13. 关键产品判断

1. `yi.wuaishare.cn` 专注东方易学工具，不混入星座、塔罗等西方占卜。
2. 站名推荐「易知阁」，「吾爱易学」作为副标题或品牌关联。
3. 首期从「五行穿衣指南查询」切入，因为它高频、轻量、适合搜索流量。
4. 五行穿衣工具本质上只有 5 套核心逻辑，但内容和视觉要通过季节、人群、场景丰富化。
5. 日更文章和工具页要并行：文章抢搜索流量，工具页做长期入口。
6. 图片素材不要一次性做 240 张，先做 20 张 Hero + 12 张春季卡片。
7. 所有内容定位为传统文化与生活参考，不做确定性承诺。
